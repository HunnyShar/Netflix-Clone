🎬 Netflix Clone

A fully responsive Netflix-inspired streaming platform built using HTML, CSS, and JavaScript. The website allows users to browse movies and TV series, watch trailers, and enjoy a dynamic landing page that changes content every time the page is refreshed.

🚀 Features
* Netflix-inspired modern UI
* Responsive design for desktop, tablet, and mobile devices
* Dynamic landing page with randomly changing featured content on every refresh
* Movies and TV series collection
* Trailer playback functionality
* Interactive hover effects and animations
* Smooth navigation experience
* Clean and user-friendly interface

🛠️ Technologies Used
* HTML5
* CSS3
* JavaScript

📚 Key Learnings
* DOM Manipulation
* Responsive Web Design
* Dynamic Content Rendering
* JavaScript Event Handling
* UI/UX Development
